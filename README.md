# Container Scanner Backend

This project provides the backend scaffold for the React Native container scanner application.

## Features

- Express.js server with TypeScript
- Supabase client configuration
- Placeholder REST routes for containers, barcodes, auth, and settings
- Centralized error handling and response helpers
- Environment-based configuration

## Scripts

- `npm run dev` - start the development server
- `npm run build` - compile the TypeScript project
- `npm start` - run the compiled production build

## Environment

Copy `.env.example` to `.env` and fill in the required values.
